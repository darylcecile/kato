# Kato

A proof-of-concept notation language — an alternative to YAML and JSON — designed to be both human and agent friendly.

The name comes from the Mauritian Creole *"kato ver"*.

## What's in this repo

- **Spec** — Language specification and grammar
- **Examples** — Sample `.kato` files
- **Parsers** — Reference implementations in Go and TypeScript

## Status

🚧 Early development — spec is evolving.
